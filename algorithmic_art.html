Python: ROYGBIV Circumferences

The algorithm:

  • Takes input parameters to compute an arbitrary number of points on the circumferences of each of two circles with arbitrary origins and radii.

  • Randomly chooses an arbitrary starting point on each circle and draws a colored line between them.

  • Proceeds around each circle, one point on each at a time, drawing colored lines connecting the circumferences.

  • The total number of lines drawn is equal to the lowest common multiple of the number of points on each circumference.
